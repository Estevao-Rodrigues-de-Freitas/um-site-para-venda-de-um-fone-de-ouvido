# UM SITE PARA VENDA DE UM FONE DE OUVIDO

---

## Descrição

Este site foi criado para emular a venda de um único produto, a ideia é focar na apresentação do produto ao máximo.
Inicialmente criado para telas de desktop e depois adaptado para telas menores.
---

## Tecnologias Utilizadas:

- HTML5  
- CSS3  
- JavaScript  
- Google Fonts
- Photoshop (para criação de imagem)
- Blender (para criação de vídeo)   

---

## Como Acessar a Interface
Link do site: https://estevao-rodrigues-de-freitas.github.io/um-site-para-venda-de-um-fone-de-ouvido/ 

---

## Explicação das Seções
### 1. Hero
- Entrega uma descriação básica do site

### 2. Itens
- Mais informações sobre o fone de ouvido e o cabo USB

### 3. Controle Sonoro
- Mostra como funciona para modificar o som dos fones de ouvido

### 4. Conector USB
- Algumas informações e alertas sobre o conector USB

### 5. Ligar/Desligar
- Demonstra como ligar e desligar o aparelho

### 6. Benefícios
- Um campo dedicado a comentar sobre a quantidade em estoque do produto e possíveis promoções.
