FONES TECNOLOGICOS

Este template emula um site criado para a venda de um único produto, nessa caso um fone de ouvido que tem um custo estimado de R$300. Este site foi criado utilizando de HTML, JavaScript e CSS sem a utilização de nenhuma biblioteca. Todas as imagens e vídeos foram criados por mim Estevão Rodrigues de Freitas utilizando photoshop e blender.

ITENS

  • Fones de Ouvido

    Este campo é dedicado a destrinchar informações sobre o fone de ouvido, indentificado o modelo do fone de ouvido, qualidade sonora, qualidade de uso do aparelho, tempo de carga e até mesmo do que o dispositivo é efeito, assim respondendo possíveis dúvidas que os clientes deteriam sobre o produto.

  • Cabo USB

    Assim como o campo acima este também serve para entregar informações sobre o produto, só que neste caso um cabo USB, indicando o tamanho do cabo e o tempo até a carga completa, com um alerta de não ultrapassar este limite de tempo, pois isto pode danificar a capacidade de carga máxima do dispositivo.

CONTROLE SONORO

Neste campo temos informações sobre o controle de som do aparelho, explicando onde estão localizados os botões que controlam o volume, a pontência minima e máxima do aparelho, demonstrada visualmente pela imagem a direita, e por meio de interatividade no site pelos botões abaixo.

TENTE

CONECTOR USB

Com este campo obtemos informações sobre onde o conector do USB se encontra no dispositivo, explicando a forma correta de encaixar o cabo no aparelho, e alertando o cliente para não tentar força a entrada do cabo pois esta ação pode danificar o fone de ouvido.

LIGAR / DESLIGAR

Este campo entrega informações sobre onde se encontra o botão de ligar e desligar do dispositivo, comentando sobre indicativos sonoros e visuais para se notar se o aparelho está ligado ou não, em conjunto a um teste visual com a imagem a direita, e um botão de interatividade logo abaixo.

TENTE

BENEFÍCIOS

Este último campo é dedicado a comentar sobre o produto de forma mais abrangente, comentando sobre possíveis promoções, quantos produtos se tem em estoque e por quanto tempo este aparelho estará a venda, ou seja, uma última via de comunicação com cliente antes de uma possível compra.
